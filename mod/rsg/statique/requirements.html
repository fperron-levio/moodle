Cette formation est conçue pour les écrans d’ordinateurs et de tablettes mesurant 10 pouces et plus sur la diagonale (résolution minimale de 1024 x 768). Nous vous suggérons d’utiliser un ordinateur de table ou un portable afin de bénéficier de façon optimale des fonctionnalités de RSG en ligne. Toujours pour apprécier pleinement votre expérience de navigation, veuillez suivre ces recommandations afin d’assurer la fluidité du visionnement des vidéos et d’écoute des éléments audios.

Navigateur recommandé :
  • Google Chrome
Navigateurs pris en charge par les ordinateurs de bureau et les portables :
  • Google Chrome
  • Internet Explorer 11 et versions suivantes / Microsoft Edge
  • Mozilla Firefox
  • Safari
Navigateur pris en charge par tous les types de tablettes :
  • Google Chrome
Connexion Internet :
  • Vitesse de téléchargement minimale de 6 Mbps (Internet moyenne vitesse)
Résolution minimale de l’écran :
  • 1024 x 768 (diagonale de 10 pouces et plus)
Processeur :
  • 1 gigahertz (GHz) ou plus rapide
Système d’exploitation :
  • Windows 10 (32 bits ou 64 bits), mise à jour Windows 8.1 (32 bits ou 64 bits), Windows 7 avec Service Pack 1 (32 bits ou 64 bits), Windows Server 2012 R2, Windows Server 2008 R2 avec Service Pack 1 (édition 64 bits uniquement), Mac OS X, IOS et Android (pour les tablettes)
Mémoire vive :
  • 1 gigaoctet (Go) pour un système de 32 bits ou 2 Go pour un système de 64 bits
Espace disque :
  • 16 Go pour un système d’exploitation de 32 bits ou 20 Go pour un système d’exploitation de 64 bits
